# scanf("%d", &a);
```&n```에서 ```&```을 사용하는 이유: ```&```을 붙이면 변수에 주소를 지정해주는 거임
* 변수가 사용되는 경우
    * 변수가 값으로 바뀌는 경우
    * 변수가 위치한 주소를 지정함(?)
* 함수 호출이될 때 값이 먼저 참조가 됨

## format specifier
|기호|설명|
|:---:|:---:|
|%d|정수형|
|%s|문자열|
|%c|문자 한 개|
|%f|실수형|
